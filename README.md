# swagger-js-doc
an implementation of parse swagger api doc into js doc, and generate api mock files.

- [ ] 扫描swagger的api接口，生成接口对应的path和method的映射 
- [ ] 生成接口请求字段，判断可选类型，请求方式
- [ ] 根据definitions生成返回字段格式
- [ ] 生成统一的前端service文件
- [ ] 根据接口信息生成js-doc
- [ ] 根据返回数据字段格式生成mock数据
- [ ] 支持单独根据某个接口生成
- [ ] 支持单独根据某个方法或某组方法来进行生成
